# Checklist de Accesibilidad ConsejoSeguro

- [ ] Todas las páginas tienen landmarks semánticos (`header`, `nav`, `main`, `footer`).
- [ ] Todos los controles interactivos tienen etiquetas y roles ARIA apropiados.
- [ ] Navegación por teclado completa y visible (focus styles).
- [ ] Contraste de color mínimo 4.5:1 en texto y elementos interactivos.
- [ ] Imágenes con `alt` descriptivo.
- [ ] Formularios con etiquetas asociadas.
- [ ] Idioma principal declarado (`lang` en HTML).
- [ ] Pruebas automáticas con axe-core y Lighthouse.
- [ ] Sin contenido animado que cause distracción o malestar.
- [ ] Instrucciones claras para usuarios de lectores de pantalla.
