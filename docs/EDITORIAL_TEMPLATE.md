# Plantilla Editorial ConsejoSeguro

Cada entrada de contenido debe incluir:

- **Título**: Breve y claro.
- **Introducción empática**: Una frase que reconozca el estado emocional/práctico del usuario.
- **Pasos claros**: Lista numerada de acciones concretas.
- **Detalles locales**: Direcciones, nombres de formularios, horarios, etc.
- **Cita oficial en línea**: Fuente oficial (nombre y URL `.gov.rs` u `officialgazette.rs`).
- **Enlaces relacionados**: Otras guías o flujos relevantes.
- **Traducción**: Campo para versión en serbio/cirílico.

---
